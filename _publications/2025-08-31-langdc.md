## Overview

Large video-language models often process a large number of visual tokens, making inference expensive for long or visually dense videos. Many existing compression methods use a fixed compression ratio, which treats static clips and information-rich clips the same way.

LangDC introduces a **Language-aware Dynamic Token Compressor**. It uses a lightweight language expert to describe video clips and converts those descriptions into soft caption tokens as visual representations. This lets the model allocate more tokens to semantically rich clips and fewer tokens to simpler clips.

## Key Ideas

- Dynamic video token compression guided by language descriptions.
- Semantic density-aware supervision for preserving visual cues needed by downstream reasoning.
- Adaptive compression ratios that follow the richness of each video segment.

## Results

LangDC reduces FLOPs by **49%** compared with the VideoGPT+ baseline while maintaining competitive performance across video understanding benchmarks. Qualitative results further show that LangDC adjusts compression based on video segment semantic density.
